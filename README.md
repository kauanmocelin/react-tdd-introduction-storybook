# React TDD introduction with Storybook

This project was developed to learn about React with Test Driven Development (TDD) and Storybook.

### Install

Install node depencies

```
npm install
```

### Get Started

Run Storybook application

```
npm start
```
Run tests with TDD

```
npm run test:tdd
```
